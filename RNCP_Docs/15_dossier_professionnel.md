# RNCP - Préparation non-technique - Dossier Professionnel (DP)

**URL :** https://intranet.hbtn.io/projects/3385
**Catégorie :** Non-technique — Dossier Professionnel
**Type d'évaluation :** Manual Review (12 pts)

---

## Contexte

Le Dossier Professionnel (DP) est un **rendu obligatoire**.
Il complète le dossier de projet pour l'entretien final avec le jury.
Il permet de justifier les compétences du titre en dehors du projet principal.

---

## ⚠️ Règles importantes

- Utiliser **impérativement le template officiel** (ne pas en créer un personnalisé)
- Faire une copie du template du titre concerné et la modifier
- Compléter la **déclaration sur l'honneur**
- Avoir **au moins 1 exemple par activité titre**
- Envoyer en **PDF** à l'équipe campus une fois terminé

---

## Contenu du DP

### Activités titre
Les activités types sont dans le **REAC** du titre visé.

### Exemples à intégrer
- Exemples de projets Holberton réalisés (HBnB, Tests, portfolios fondamentaux...)

- Tout projet GitHub pertinent accessible le jour J

> 💡 **Conseil** : Si votre projet RNCP est léger sur une compétence, compensez dans le DP avec d'autres projets ou expériences.

### GitHub
- S'assurer que les projets GitHub pertinents sont **accessibles publiquement** le jour J
