## Projets éligibles au RNCP

Le projet doit être une **application web** (ou mobile + web).
Exemples éligibles : application mobile et web, application web seule.
**Non éligibles** : jeu 2D seul, algorithme de compression, IA seule.

> Si l'application n'est pas déployée : avoir une démarche technique documentée (diagramme de déploiement UML, explication détaillée dans le rapport).

## Blocs de compétences

Il est possible de ne valider qu'**un seul bloc** — les blocs échoués peuvent être repassés dans un délai d'un an.
