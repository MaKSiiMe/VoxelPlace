| **REV** | Référentiel d'Évaluation |

---

## Format de l'épreuve

- **Durée** : 30 minutes
- **Ressources autorisées** : uniquement la **documentation fournie pendant l'épreuve** (pas de docs personnelles, pas d'internet)
- **Structure** :
  - 1 question fermée (QCM) en **anglais**
  - 1 question ouverte en **anglais**
  - 1 question fermée (QCM) en **français**
  - 1 question ouverte en **français**

## Exemples de questions types

- En vous référant à la documentation technique : quel outil n'est pas un besoin dans un environnement de développement ? (Git / Linux / NodeJS)
- Quelle instruction git permet d'ajouter de nouveaux fichiers ? (`mkdir` / `git push` / `git add`)
- What features does Github provide?
- What is the major/central concept of Node.js?
